# AI-Powered Contact Form Responder

## What it does
Automatically responds to contact form submissions with a personalized, 
AI-generated email reply.

## Workflow
Form Submission → Google Sheets → Google Gemini AI → Gmail

## Features
- Saves all form submissions to Google Sheets
- Uses Google Gemini AI to generate personalized email replies
- Automatically sends the reply via Gmail

## Tools Used
- n8n
- Google Sheets
- Google Gemini API
- Gmail

## Use Case
Perfect for small businesses, freelancers, and agencies who want to 
automate their contact form responses intelligently.
